# HULLAH – Outfit Recommendation System

HULLAH is an intelligent outfit recommendation system that uses AI, machine learning techniques, and fashion datasets to generate personalized outfit suggestions based on user preferences and body characteristics.

## Technologies Used

* Python
* Flask
* HTML / CSS / JavaScript
* Pandas
* Scikit-learn
* TF-IDF
* Cosine Similarity

## Run Locally

1. Download or clone the project files into one folder.

2. Make sure all project files are located in the same directory with `app.py`.

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open the browser and visit:
   http://127.0.0.1:5000

## Features

* Personalized outfit recommendations
* AI-based recommendation system
* Fashion dataset processing
* User feedback system
